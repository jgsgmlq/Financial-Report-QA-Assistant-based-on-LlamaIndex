\section{Abstract}

This document provides a detailed introduction to the design and implementation of the Insight: Financial Report Analysis Engine project. This project is a vertical domain RAG (Retrieval-Augmented Generation) question-answering system based on LlamaIndex, focusing on the analysis of long, information-dense financial PDF reports, and providing page-level citation tracking.

The system adopts a highly cohesive, low-coupling modular design, consisting of four core components: data ingestion layer, retrieval and generation layer, evaluation closed-loop layer, and interaction layer. Through a two-phase development strategy, the system achieved a gradual upgrade from basic baseline to advanced features, including semantic chunking, hybrid retrieval, RAPTOR tree-based summarization, and other advanced technologies.

This report elaborates on the system's architectural design, core module implementation, evaluation methods, and experimental results, demonstrating the application value and technological innovation of the system in the field of financial report analysis.
